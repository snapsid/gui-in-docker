# GUI IN DOCKER

Running GUI Apps inside Docker Container

### Installation:
Clone the Repo
Install docker

### Run docker service (below comand is for redhat linux)

```
$ systemctl start docker
```

### Making Script runnable

```
$ chmod +x gui.sh
```

### Run the script

```
$ ./gui.sh
```

### enter os name

```
$ os1
```

### enter image name

```
$ centos:latest
```
